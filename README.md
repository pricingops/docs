## Setup

Check the `nuxt.config.ts` file for important configuration details. It includes information about plugins, modules, and environment variables that may need to be set up for the project.

Install dependencies:

```bash
yarn install
```

## Deployment

1. Development mode: Run `yarn dev`. This starts the development server.
2. Build: Run `yarn build`. This builds your application for production.
3. Generate: Run `yarn generate`. This generates your static application.
4. Preview: Run `yarn preview`. This previews the result of your build locally.

For a detailed explanation of how things work, check out [Docus](https://docus.dev).
